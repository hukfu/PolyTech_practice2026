📖 О проекте «project SpaceWolf»

Актуальность проекта

Отечественная игровая индустрия активно развивается. Несмотря на технический прогресс, игровое сообщество всё чаще сталкивается с проблемой однообразия игрового опыта. В сегменте космических экшн-игр наблюдается определенная стагнация: большинство проектов либо следуют жесткой «рельсовой» структуре, либо предлагают чрезмерную свободу в пустом пространстве, лишенную внятной динамики. Актуальность данного проекта заключается в создании продукта, который преодолевает эти жанровые шаблоны через инновационное проектирование уровней и уникальную визуальную подачу.

Проблематика проекта: На данный момент космические шутеры являются крайне непопулярным жанром. Проблема современных научно-фантастических шутеров — их структурная предсказуемость. Игрокам не хватает проектов с глубокой проработкой игровых ситуаций, где каждый этап прохождения предлагает новый темп и задачи.

🛠️ Технологический стек

  • Движок: Unity 2022 LTS
  • 3D-моделирование: Blender
  • Версионирование: Git + GitHub (Git LFS)

Ключевые особенности

01 Демо
Полнофункциональная демо-версия космического шутера
02 Механики
Механика переключения между кораблями в реальном времени, расширяющая тактические возможности игрока
04 Сражения
В демо-версии будет представлен босс-файт: сражение с финальным врагом
03 Графика
Свой уникальный визуальный стиль: градиентное текстурирование

🤖 Моя роль: 3D-художник

В проекте я отвечаю за полный цикл создания 3D-моделей игровых кораблей:

  • Разработка внешнего вида корабля
  • Низкополигональное моделирование (Blender)
  • UV-развертка и текстурирование
  • Экспорт и настройка материалов для Unity

На данный момент полностью готова модель флагманского корабля и затекстурировано около десяти дополнительных моделей.
